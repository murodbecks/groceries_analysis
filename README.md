## Setup and Installation

All scripts are in [groceries_analysis.ipynb](groceries_analysis.ipynb) Jupyter Notebook. It is designed to be run top to bottom and reproducible. So you can run it both locally or in [Google Colab](https://colab.research.google.com/).

Initial data ([groceries.csv](data/groceries.csv)) and derived data ([product_prices.csv](data/product_prices.csv) and [transactions_priced.csv](data/transactions_priced.csv)) are located in `data/` folder. 

### Local Usage:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/murodbecks/groceries_analysis.git
   cd groceries_analysis/
   ```

2. **Create and activate a virtual environment (recommended):**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. Run [groceries_analysis.ipynb](groceries_analysis.ipynb) top to bottom.
